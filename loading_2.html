Loading your query genomes

Tool 2 requires two inputs: a file containing the subject sequences (<customSeq>_sequences.fa), and the query genome assemblies. These must be saved in the correct input folders:

  • save the subject sequence file in input_custom_seqs/ folder.
  • save the query genome assemblies in Tool 2â€™s input_genomes/ folder.

Preparing the custom sequences is described here.

The assembled genomes can be found in the LIBRARY folder.
